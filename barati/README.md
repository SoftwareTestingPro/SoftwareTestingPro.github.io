# Barati - Your Chosen Wedding Family

Connect with a supportive community for your big day. Find family members, relatives, and friends to perform wedding rituals and make your celebration memorable.

## Live App
You can access the live web version here:
👉 **[Launch Barati App](https://softwaretestingpro.github.io/barati/app/)**

## Project Overview
This is a multi-platform Flutter application (Android, iOS, and Web) integrated with Supabase for real-time synchronization.
