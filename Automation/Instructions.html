INSTRUCTIONS
  • This platform overcomes the challenges faced by software automation learners where they could not find a practice platform to automate different kinds of scenarios.
  • You can use this platform using any programming language and any software automation tool like Selenium, Cypress, Playwright, Test Complete, etc.
  • Create a new project in your preferred IDE.
  • Add packages for each proficiency level, i.e., Beginner, Intermediate, Advanced.
  • Maintain each automation test in its respective package.
  • Select any task as per your current software automation knowledge and automate it.
  • Most of the tasks display a success message when successfully automated. Assert it.
  • For mobile automation you can automate the scenarios in Mobile devices or emulators/simulators.
  • For any challenges faced, search for solutions on the internet.
