Case 6: Provider Performance

Which clinic specialties are most expensive? Calculate average payout per specialty.

SELECT p.specialty, AVG(c.paid_amount)
FROM dim_providers p
JOIN fact_claims c ON p.provider_id = c.provider_id
GROUP BY 1 ORDER BY 2 DESC;

Case 7: Audit History Trail

Identify claims that have undergone status changes. Perfect for practicing CTEs.

WITH history_counts AS (
  SELECT claim_id, COUNT(*) as changes
  FROM fact_claim_status_history
  GROUP BY 1
)
SELECT * FROM history_counts WHERE changes > 0;
